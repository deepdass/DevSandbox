# DevSandbox
A Development Sandbox made in unreal engine using both c++ and blueprint
<br> Plannig to make this into a sandbox which will contain my game dev shenanigans

## Why
I wanted to get good at writing modular and component-based architecture in C++ not just inheritance chains, with Unreal replication/ multiplayer while being performant

Controls


## what it has

Action System - similar to Gameplay Ability System (GAS) with data objects called action having ability which can be given to any actor having action component. Uses Gameplay Tags and modular design - Credit Tom Looman's Design Followed [Github Repo](https://github.com/tomlooman/ActionRoguelike)
